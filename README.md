# GED Project

## Phase 1: Initial Setup

This project contains backend and frontend applications for a document management system (GED).

### Folder Structure

- backend: Contains the Express.js backend application
- frontend: Contains the React frontend application using Vite and TypeScript
- db-scripts: Contains PostgreSQL database schema and setup scripts

### How to run

- Backend:
  - Navigate to backend folder
  - Run `npm install`
  - Start the server with `npm run dev` (after creating server code in later steps)

- Frontend:
  - Navigate to frontend folder
  - Run `npm install`
  - Run the dev server with `npm run dev`
